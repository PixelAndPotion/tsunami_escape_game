# tsunami_escape_game
a python based simulation betting game where players need to escape a tsunami wave before it crashes. This game demonstrates gaming logic, GUI development and probability 
key features include: multiplayer wave that increases, probability-based crash system, simulated AI players, funny messages to players, GUI version using Tkinter, CLI version for logic testing and a betting system with balance tracking
How does the game work? The multiplier (wave) increases, the longer a player plays, the higher their chances are of winning but risk also increases, the skill events give players a chance to reduce risk and a player decides when to cash out before a wave crash
Technologies used: Python and Tkinter for the GUI
Project structure: tsunami_escape.py (CLI) , tsunami_escape_gui.py (GUI)
Purpose: This game demonstrates problem solving, GUI development and an understanding of probability-based systems
Disclaimer: No real money or gambling is used
How to run? Run the CLI and GUI
